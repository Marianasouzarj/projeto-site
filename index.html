Curso de JavaScript

JavaScript é uma linguagem de script criada para aumentar a interatividade entre um site e seus visitantes. Os códigos em JS são executados dentro do navegador e dão acesso aos elementos HTML por meio do DOM.

Acesse o curso de JavaScript

Para que serve HTML, CSS e JS?

HTML e CSS não são linguagem de programação. Programação envolve estruturas especializadas que dependem do uso de variáveis simples e compostas, condições, laços e até coisas mais complexas como objetos. Todas essas características estão presentes na linguagem JavaScript.
Essa sim é uma Linguagem de Programação.
